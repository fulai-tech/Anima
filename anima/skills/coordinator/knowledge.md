# Coordinator Skill — Cross-Device Orchestration Knowledge

## Device Interactions
- AC + Humidifier: AC dries air → increase humidifier target when AC turns on
- AC + Windows: open windows make AC wasteful → alert user
- Heater + Humidifier: heating dries air → same as AC
- Light + Time: align all lights with circadian rhythm
- Purifier + Windows: open windows reduce purifier effectiveness

## Orchestration Principles
- Avoid conflicting actions (e.g., heating + cooling simultaneously)
- Energy-first: prefer passive solutions before active devices
- Gradual: coordinate changes across devices over 5-10 min, not all at once
- Priority: safety > comfort > energy saving
